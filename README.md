# 关于本项目

我之前在网上看到过那种能把那些治愈的话语填满屏幕（满屏弹窗）的代码，于是用自己现有的技术也做了一个，要的就是满屏幕的安全感.
如果能帮到你的话，我很高兴.

## 用法

第一个输入框用于设置弹出条数（0 ~ 999），如果设为 0 则是无限弹出（不过我用了点特殊方法，让弹出过多的时候不至于卡顿）.
下面的多选框则是选择类型，目前有 7 种：

- Acception（无条件接纳）
- Accompaniment（陪伴）
- Affirmaion（自我价值肯定）
- Appreciation（特质欣赏）
- Encouragement（鼓励行动）
- Self-Compassion（自我关怀）
- Soothing（情绪安抚）

## 下载、运行、修改

（假设你装了 Git 和 Node.js）
下载到本地：

```bash
git clone https://github.com/docccccccccccc/FillMyScreen.git
```

然后切换到项目目录，运行下面的命令：

```bash
npm install
npm run dev
```

至于修改——用编辑器打开项目即可.
